# AI & Data Job Market Intelligence System

## Overview
This project analyzes real-world AI and data-related job market trends using live API data.

The project includes:
- ETL pipeline
- SQL analytics
- NLP-based skill extraction
- Salary analysis
- Power BI dashboard
- Business intelligence insights
